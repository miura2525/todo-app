# 課題管理 (ToDo) アプリケーション

ToDoリストを簡単に管理・操作できるデスクトップアプリケーションです。

## 1. 概要
- 課題（タスク）の追加、削除、完了チェック、並べ替えができるアプリです。
- 期限や内容を直感的に管理できます。

## 2. 実行環境（必要な動作条件）
- Python 3.x がインストールされていること

## 3. 起動手順・動かし方

### ① ダウンロードと解凍
GitHubの `Code` ➔ `Download ZIP` からZIPファイルをダウンロードし、解凍します。

### ② フォルダへの移動
コマンドプロンプトやターミナルを開き、`todo_app.py` が保存されているフォルダへ移動（`cd`）します。

```cmd
cd フォルダのパス（例: cd Desktop\todo-app）

```bash
python todo_app.py
