# 課題管理 (ToDo) アプリケーション

ToDoリスト（課題やタスク）を簡単に管理・操作できるデスクトップアプリケーションです。

---

## 概要
本アプリケーションは、課題の追加、締切日によるソート（並べ替え）、完了・削除といった機能を直感的に操作できるGUIツールです。

---

##  実行環境
* **Python**: 3.x 以降
* 特別な外部ライブラリの事前インストールは不要です（Python標準ライブラリで動作します）。

---

##  起動・実行手順

### 1. リポジトリの取得
GitHub画面の右上にある **`Code`** ボタン ➔ **`Download ZIP`** をクリックしてファイルをダウンロードし、任意の場所へ解凍します。  
*(Gitが使える環境の場合は `git clone <リポジトリURL>` でも取得可能です)*

### 2. 作業フォルダへ移動
コマンドプロンプト（Windows）またはターミナル（Mac）を開き、`todo_app.py` が入っているフォルダへ移動（`cd`）します。

### 3.実行
cdで移動後以下を実行してください。

```bash
python todo_app.py
