# 課題管理 (ToDo) アプリケーション

ToDoリスト（課題やタスク）を簡単に管理・操作できるデスクトップアプリケーションです。

---

##  1. ソースコードの構成

```text
todo-app/
├── task.py                # 【Entity】Task / TaskList クラス（課題データ・管理ロジック）
├── create_task_control.py # 【Control】 CreateTaskControl クラス（入力検証・ソートロジック）
├── sub_screens.py         # 【Boundary】各種ダイアログ画面（詳細確認・削除・状態確認）
├── main_screen.py         # 【Boundary】HomeScreen クラス（メインGUI画面）
├── main.py                # アプリケーション起動用エントリーポイント
└── README.md              # 取扱説明書（本ファイル）
```

---

##  2. プログラムの起動・コンパイル方法

### 必要な環境・パッケージ
* **Python**: 3.x 以降
* **外部パッケージ**: 不要（Python標準ライブラリ `tkinter`, `datetime` のみで動作するため、コンパイルや `pip install` 等の事前準備は不要です）

### 起動手順

1. **リポジトリの取得（ダウンロード）**  
   GitHubの `Code` ➔ `Download ZIP` 等でファイルを取得し、解凍します。

2. **作業フォルダへ移動**  
   コマンドプロンプト（Windows）またはターミナル（Mac）を開き、`main.py` が存在するフォルダへ移動（`cd`）します。
   ```cmd
   cd フォルダのパス
   ```
   *(例: `cd "C:\Users\ユーザー名\Desktop\todo-app"`) *

3. **プログラムの実行**  
   エントリーポイントである `main.py` を指定して実行します。
   ```cmd
   python main.py
   ```
   *(※環境によっては `python3 main.py` または `py main.py`)*

##  3. プログラムの操作方法（アプリの使用方法）

### ① 課題の新規追加
1. 画面上部の「新規課題の追加」フォームにて、以下の項目を入力します。
   * **課題名**: 20文字以内（必須）
   * **締切日**: `YYYY-MM-DD` 形式（初期値は本日の日付）
   * **課題内容**: メモや詳細説明（任意）
2. **「追加」** ボタンを押すと、一覧に追加されます。

### ② 課題一覧の確認・強調表示
* 登録された課題は、中央の一覧エリアに「課題名・締切日・状態（未完了/完了）」が表示されます。
* **強調表示（ハイライト）**: 未完了の課題のうち、**提出期限まで残り2日以内** のものは背景色がピンク（赤字）で強調表示されます。

### ③ 締切日順の並べ替え
* **「締切日の昇順で並べ替え」** ボタンをクリックすると、締切日が近い順に一覧が自動で整列します。

### ④ 課題詳細の確認
* 一覧から確認したい課題を選択し、下部の **「詳細を確認」** ボタンを押すと、ポップアップ画面で入力した「課題内容」を確認できます。

### ⑤ 完了 / 未完了の切り替え
* 課題を選択し、**「完了 / 未完了 に変更」** ボタンを押すことで、ステータスを交互に切り替えられます。

### ⑥ 課題の削除
* 課題を選択し、**「削除」** ボタンを押すと、確認ダイアログが表示され、確定すると課題が削除されます。
